# codewithmohan · Portfolio

> Interactive developer portfolio for [@codewithmohan0](https://github.com/codewithmohan0) — powered by the GitHub API.

---

## 🚀 Features

- **Live GitHub Data** — auto-fetches repos, bio, followers & stars on load
- **Projects Grid** — all public repos with language badges, stars & forks
- **Contribution Heatmap** — visual activity graph with hover tooltips
- **Terminal UI** — stylized `whoami` display of your GitHub profile
- **Animated Stats** — counters for repos, followers, following & total stars
- **Custom Cursor** — smooth trailing cursor effect
- **Scroll Reveal** — staggered animations as you scroll

## 🛠 Tech

Single-file HTML · Vanilla JS · GitHub REST API · Google Fonts (Syne + Space Mono)

## 📦 Usage

```bash
# Just open in any browser — no build step needed
open index.html
```

Or host it on **GitHub Pages**, **Netlify**, or **Vercel** by dropping the file into your repo.

## ⚙️ Customization

| Variable | Location | What it changes |
|---|---|---|
| `USERNAME` | `<script>` | GitHub username to fetch |
| `--accent` | `:root` CSS | Primary neon color |
| `--accent2` | `:root` CSS | Secondary color |
| `hero-tag` text | HTML | Tagline under nav |
